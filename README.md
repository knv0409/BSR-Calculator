## 1. 성장 재화 계산하기

기존 재화 계산기들이랑 비슷함. 성장 계산할 캐릭터 추가하고 내가 가진 재화 입력하면 계산이 되는 구조임

![1](https://github.com/user-attachments/assets/f0526c39-da47-4fc4-adea-b60799e46158)


### 🖼️ 배경 화면

프로그램을 처음 켜면 웰컴젖통이 널 반길거임. 보기 거슬리면 위에 배경 표시 버튼 누르면 사라짐. 배경은 13장 정도 있음. 사실 배경 넣는게 제일 쓸데없는 짓인데 내 쥬지가 나에게 넣으라고 시킴

빨간색 동그라미친 부분을 이용해 캐릭터를 추가하셈


![2](https://github.com/user-attachments/assets/d8b79b3c-f131-46dd-916f-48e1a79dcb1d)


### ⚙️ 캐릭터 설정

일단 기본 목표값은 모든 레벨, 스킬, 각인 올 만렙임


마유리처럼 봉멸이나 일반공격 안찍어도 되는 애들은 **스킬/각인 설정** 버튼 눌러서 알아서 세팅하면됨

소이퐁은 전술 캐릭이기 때문에 **5. 전장스킬**까지 잘 반영되어있음

> 각인 30레벨은 세트각인+핵심각인 만렙을 합쳐놓은거임. 각인은 레벨별로 나눠 성장할 일이 별로 없어보여서 계산안하거나 만렙이거나 둘 중 하나임



![3-2](https://github.com/user-attachments/assets/5078ad32-6269-41b3-af09-c527c08976a5)



### 📦 재화 입력

재화입력창에 재화를 입력하면 됨

종류가 ㅈㄴ 많아서 귀찮은 경력, 요령, 접쇠, 망치, 오의는 인게임 내 인벤토리 순서대로 정렬해놨기 때문에 오른쪽 위의 **'항상 위에 고정'** 을 켜놓고 게임화면과 같이 보면서 입력하면 됨

> 숫자 입력창은 키보드 방향키를 이용해 위아래 좌우로 전환이 가능하기 때문에 마우스로 일일히 찍지 말고 키보드로 입력하셈
>
> 


![4](https://github.com/user-attachments/assets/49f807b3-261f-44f1-b6c2-f7d1bfa0ab57)


### 📊 결과 및 파밍 플랜

3번 탭의 결과창의 초록버튼을 누르면 계산결과가 나옴

물자 던전 레벨과 대행 횟수를 선택하고나서 **'대행 실행'** 을 누르면 그만큼의 재화가 즉시 반영됨

> 이벤트로 거의 지급하지 않는 요령, 망치, 오의, 오마모리 같은 경우 이 기능을 이용하면 편하게 쓸 수 있음




## 2. 패키지 효율 계산

![5](https://github.com/user-attachments/assets/329f4c0f-eb36-4eec-9267-60be9b45ee04)


가격, 영옥, 티켓 이외는 계산하지 않음. 상시로 판매하는건 프리셋으로 만들어놓음


![7](https://github.com/user-attachments/assets/1bbc0e22-493c-4dbc-93e3-421155562ac3)


캐릭터 추가하듯이 적으라는거 적고 파란 버튼 누르면 그 패키지 효율이 얼마인지 하이라이트됨


![8](https://github.com/user-attachments/assets/c728afba-1bbf-4238-af9d-2eb27a2b5f13)


예시로 마유리 출석 패키지를 추가해보면 **5.08배 효율**이라고 뜸. 인게임에서는 800%라고 하던데 영력약 같은게 포함된 가격인듯

## 3. 가챠/과금 플래너


![9](https://github.com/user-attachments/assets/2922bafe-3070-4b52-af68-9fb1d2120514)


내가 가진 재화 정보를 바탕으로 나중에 한정 캐릭을 뽑을 때 얼마나 현질해야할지 적당히 계산해주는 탭

> ⚠️ 가챠겜이 다 그렇듯 걍 운빨좆망겜이기 때문에 계산기의 결과를 너무 맹신하지 말것

- 평균기댓값은 캐릭 **54뽑**, 무기 **31뽑** 기준



![10](https://github.com/user-attachments/assets/4246c9a7-e9e3-4c40-b7f5-5d56e8feba46)


### 💰 재화 입력

재화를 입력하고 빨간글씨를 읽어보고 해당되면 체크하면 됨

![12](https://github.com/user-attachments/assets/7ba198ea-1287-4761-a944-4105d51dcdbe)

- **첫번째 빨간 글씨 (확인서 교환)** — 매월 주는 확인서 교환을 이미 받았는지 물어보는거임
- 

![11](https://github.com/user-attachments/assets/5427e20b-9601-41cc-bf93-bd565f52520a)

- **두번째 빨간 글씨 (픽업 출석 티켓)** — 픽업마다 기본으로 주는 티켓 7장짜리를 받았는지 물어보는거임




![13](https://github.com/user-attachments/assets/b68f82ae-22df-4cce-9fa9-5cb1747431c8)


### 🎯 픽업 타임라인

파란버튼을 누르면 자동으로 픽업기간이 추가됨. 픽업 날짜는 자동으로 3주 간격으로 맞춰주기 때문에 거의 바꿀 필요 없을거고 몇 돌 할지만 정하면됨

픽업을 추가했으면 **'마지막 픽업 시작일로 자동 맞추기'** 버튼을 누르셈. 그럼 오늘부터 마지막 픽업 시작일이 목표일로 자동 설정됨



![14](https://github.com/user-attachments/assets/3594b066-c94d-439b-87d1-9c10d982b1ff)


### 🛒 과금 패키지 플랜

상시로 판매하는 패키지는 자동으로 계산할 수 있도록 만듬. 각 항목을 체크하면 목표 기간에 맞춰서 자동으로 구매횟수가 입력됨

> 월간 패스의 경우 패스 남은 일수를 입력해줘야 계산이 보다 정확해짐. 이 부분은 귀찮으면 생략해도됨




![15](https://github.com/user-attachments/assets/cbf191a3-3d4c-49e6-a130-4feb4e2fa745)


### 📈 계산 결과

할거 다하고 커다란 초록버튼을 눌러서 계산하면 결과가 뜸

무료수급 부분에서 좀 오차가 있을 수 있는데 일단 사투+협토=1주일에 750 영옥으로 계산했고 이벤트는 3주마다 3500영옥으로 대충 계산함

> 이벤트는 얼마나 주는지 알 수가 없고 이벤트마다 주는 양이 다르기 때문에 아마 여기서 오차가 클거임
